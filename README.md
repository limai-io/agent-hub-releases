# Agent Hub CLI Releases

Download binaries from the [latest release](https://github.com/limai-io/agent-hub-releases/releases/latest).

## Install

```bash
curl -fsSL https://github.com/limai-io/agent-hub-releases/releases/latest/download/agent-hub-darwin-arm64 -o /usr/local/bin/agent-hub && chmod +x /usr/local/bin/agent-hub
```
